# OSric's Dungeon Architect

A procedural dungeon map generator compliant with AD&D 1e stocking rules, powered by Gemini AI.

## Local Development

1. `npm install`
2. `export API_KEY=your_key_here` (Linux/Mac) or `set API_KEY=your_key_here` (Windows)
3. `npm run dev`
